# arXiv Upload Checklist

Use the neutral preprint source file:

- `paper_arxiv.tex`

Upload these figure files with it:

- `robot.JPG`
- `simulation.JPG`
- `robot_shadowing_human_inrealtime.png`
- `robot_grabbing_cube.JPG`
- `chart_latency.png`
- `chart_pertile.png`
- `inthewild_humanhandhandcapture.JPG`
- `inthewild_robotIK.JPG`
- `mosaic_in_the_wild.png`

Do not upload these IEEE Access-specific files for arXiv:

- `paper.tex`
- `ieeeaccess.cls`
- `spotcolor.sty`
- `logo.png`
- `notaglinelogo.png`
- `bullet.png`
- all `t1-*.pfb`
- all `t1-*.tfm`
- all `*.map`
- all `*.fd`

Notes:

- The bibliography is embedded directly in `paper_arxiv.tex`, so no `.bib` or `.bbl` file is needed.
- `paper_arxiv.pdf` is useful for your own preview, but upload the source package to arXiv rather than only the PDF.
- On arXiv, select the top-level file `paper_arxiv.tex` if prompted.
